[LocalWP](https://localwp.com/) is totally FREE and can install more than one WordPress with it.  
Besides the local WordPress, it has many other features (which I don't really need).  
I used LocalWP to build projects in local environment and after it's done, I migrated everything to the production with [All-in-One WP Migration](https://wordpress.org/plugins/all-in-one-wp-migration/).  
It became extremely easy to work on all WordPress projects now with these 2 tools.
